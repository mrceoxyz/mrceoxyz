## 🏠 Currently Building — HOMEOS

> **Smart Home Automation for the Nigerian Market**
> Real-time IoT control, offline-first architecture, and solar/inverter
> monitoring — designed around Nigerian infrastructure realities.

- 🔌 **Hardware**: ESP32 relay modules + Raspberry Pi 4/5 as local MQTT broker
- 📡 **Protocol**: Mosquitto MQTT with WebSocket bridge for real-time device control
- 🌐 **Frontend**: Next.js 14 + TypeScript + Tailwind CSS — mobile-first dashboard
- 🗄️ **Backend**: PostgreSQL + Prisma ORM, Node.js API layer
- ⚡ **Edge Design**: Offline-first — works even when NEPA takes light
